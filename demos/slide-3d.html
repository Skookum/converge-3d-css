state change

$('.slider').addClass('active');

Using GPU

By giving an element a 3d transformation, you're strongly suggesting to the browser that it should be handled by the GPU. It will be given its own layer in the GPU and, once rendered, that layer will be composited with the other layers for painting the page.

Why not just load everything into the GPU then? Every layer in the GPU consumes memory, so eventually you'll run out. It's best to be selective about which layers you hardware-accelerate.
