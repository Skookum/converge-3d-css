normal state

active state

left: -100%;

state change

$('.slider').addClass('active');

A little better

This is what most developers do these days. They know that jQuery's tweened animations are terrible, and that CSS should control visual transitions. It's a natural step to start by animating the 'top' and 'left' CSS properties.

While this is much better than using JavaScript to tween CSS properties, it's not nearly as smooth as using 2d or 3d CSS transforms. This becomes especially evident when animating several elements simultaneously.
